# Preview Hub Design

## Goal

Create a branded landing page at `/` for `preview.thatsoftwarehouse.com` that explains the purpose of the preview subdomain, links back to `thatsoftwarehouse.com`, and keeps `/atx-dentistry` available as a live sample preview.

## Approved Direction

- `/` becomes the preview hub landing page
- `/atx-dentistry` remains the sample website preview
- Primary CTA links to `https://thatsoftwarehouse.com`
- Secondary CTA and featured card link to `/atx-dentistry`

## Visual Direction

- Darker product-style TSH surface for the hub
- Distinct from the dental preview but clearly part of the same ecosystem
- Concise copy, one strong CTA, and a single featured preview example

## Sections

1. Compact nav
2. Hero with primary and secondary CTAs
3. Use-case proof strip
4. Featured preview card
5. Simple footer attribution

## Constraints

- No new dependencies
- Static frontend only
- Keep Netlify SPA setup unchanged
- Maintain a clean path to more previews later
